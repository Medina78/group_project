# StudyFlow - OpenAI AI Assistant Setup Guide

Your AI Assistant is now configured to work with **OpenAI's API**.

## Quick Start

### Step 1: Get Your OpenAI API Key
1. Visit https://platform.openai.com/api-keys
2. Sign in or create an account
3. Click "Create new secret key"
4. Copy the key (starts with sk-)

### Step 2: Add API Key to StudyFlow
1. Go to the AI Assistant page
2. Paste your API key in the "OpenAI API Key" field
3. Select a model (GPT-3.5 Turbo recommended)
4. Click "Save Settings"

### Step 3: Start Chatting!
Type your question and click Send. That's it!

## Important Notes
- Your API key is stored ONLY in your browser
- Keys are never sent to our servers
- Check OpenAI's pricing: https://openai.com/pricing
- Clear browser data to remove stored keys

## Supported Models
- GPT-3.5 Turbo (fastest, cheapest)
- GPT-4 Turbo (balanced)
- GPT-4 (most capable)

Estimated cost per conversation: $0.001 - $0.01
